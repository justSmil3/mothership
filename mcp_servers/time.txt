"A Model Context Protocol server that provides time and timezone conversion capabilities. This server enables LLMs to get current time information and perform timezone conversions using IANA timezone names, with automatic system timezone detection.\nget_current_time\n - Get current time in a specific timezone or system timezone.\nRequired arguments:\ntimezone\n (string): IANA timezone name (e.g., 'America/New_York', 'Europe/London')\nconvert_time\n - Convert time between timezones.\nRequired arguments:\nsource_timezone\n (string): Source IANA timezone name\ntime\n (string): Time in 24-hour format (HH:MM)\ntarget_timezone\n (string): Target IANA timezone name\n"
